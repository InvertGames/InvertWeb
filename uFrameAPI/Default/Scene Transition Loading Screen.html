Scene Transition Loading Screen
Getting Started > Standard Graphs > Scene Managers > Scene Transitions > Scene Transition Loading Screen

uFrame ships with a predefined loading screen that consist of nothing more than a view.  This view has a view-model that is directly connecting to the loading/transition mechanism of uFrame.  The LoadingScreenView in the below image is a single object that has the pre-made UnityGUILevelLoaderView.  The "LoadingScreenObject" is a Unity component that does nothing more than mark the object for removal when the loading of a transition has completed.

If you want to create your own custom loading screen with a different loading screen you must create a class that derives from LevelLoaderView and implement the methods as shown below.

public class DebugLoaderView : LevelLoaderView
{
    protected override void ProgressChanged(float progressValue)
    {
        base.ProgressChanged(progressValue);
        Debug.Log("Progress has changed to: " + progressValue);
    }
    protected override void StatusChanged(string statusMessage)
    {
        base.StatusChanged(statusMessage);
        Debug.Log("Status has changed to: " + progressValue);
    }
}
