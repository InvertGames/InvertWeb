Registered Type Mappings
Getting Started > General uFrame Architecture > Dependency Injection > Registered Type Mappings

Type mapping allow you to specify types that should resolve to new instances of an interface.  So each time they are resolved or injected a new instance will be created.  If there is a registered instance it will return that instead.

RegisterTypeMappings
Copy Code 
public class SampleSceneManager : SceneManager
{
    public override void Setup()
    {
        base.Setup();
        // Registers the facebook messenger as the default type for IMessenger 
        Container.Register<IMessenger,FacebookMessenger>();
        var result = Container.Resolve<IMessenger>();
        Debug.Log(result.GetType().Name); // Results as "FacebookMessenger"
        Container.Register<IMessenger,FacebookMessenger>("Facebook");
        result = Container.Resolve<IMessenger>("Facebook");
        Debug.Log(result.GetType().Name); // Results as "FacebookMessenger"
        Container.Register<IMessenger,YahooMessenger>("Yahoo");
        result = Container.Resolve<IMessenger>("Yahoo");
        Debug.Log(result.GetType().Name); // Results as "YahooMessenger"
    }
}
