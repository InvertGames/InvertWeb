GameManager

The Game Manager has the longest life-cycle of all the uFrame classes, it will remain throughout the entire lifecycle of a game/application. It uses the Unity "DontDestroyOnLoad" functionality allowing it to persist from scene to scene.

The game managers main responsibility is to maintain the current Scene Manager and process the appropriate setup required by the rest of uFrame. The current GameManager instance can always be accessed from anywhere via its static GameManager.Instance property.

The game manager also has a static ActiveSceneManager property that can be used to access the current Scene's Manager class.
