Computed Property Triggers
Getting Started > State Machine Graphs > Using A State Machine > Computed Property Triggers

Computed Properties can be used to trigger transitions on a statemachine, their implementation is somewhat unique, the computed must be a boolean value for it to be a state machine trigger.

When a computed property trigger is recalculated, if it returns true, the trigger will be invoked. The computed will also be recalculated automatically when a state is entered.
