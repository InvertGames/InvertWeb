Element Inheritance
Getting Started > Subsystem Graphs > Element Inheritance

Inheritance is a great way to make your classes maintainable and extensible.  With uFrame you can express this via the input and output connectors of the element nodes. For Example examine the following image:

Dog and Cat both derive from Animal.  This means that the generated controller for Dog and Cat will derive from AnimalController, and the ViewModel will derived from AnimalViewModel.  So anything you implement inside AnimalController, or AnimalViewModel will also be a part of Dog, and Cat as well. So here is what it looks like in the generated code.

GeneratedElementInheritance
Copy Code 
public abstract class DogControllerBase : AnimalController {
   
    public abstract void InitializeDog(DogViewModel dog);
   
    public override ViewModel CreateEmpty() {
        return new DogViewModel(this);
    }
   
    public virtual DogViewModel CreateDog() {
        return ((DogViewModel)(this.Create()));
    }
   
    public override void Initialize(ViewModel viewModel) {
        base.Initialize(viewModel);
        this.InitializeDog(((DogViewModel)(viewModel)));
    }
}
public abstract class AnimalControllerBase : Controller {
   
    public abstract void InitializeAnimal(AnimalViewModel animal);
   
    public virtual AnimalViewModel CreateAnimal() {
        return ((AnimalViewModel)(this.Create()));
    }
   
    public override void Initialize(ViewModel viewModel) {
        this.InitializeAnimal(((AnimalViewModel)(viewModel)));
    }
}
