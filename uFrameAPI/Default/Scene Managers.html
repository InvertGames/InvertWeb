Scene Managers
Getting Started > Standard Graphs > Scene Managers

Scene Managers ultimatley provide context to a scene defining how it loads, unloads and transitions to other scenes.  It is maintained by the Game Manager and only one will be active at a time. 

Every scene manager requires an appropriate sub-system to be attached to it.  This will define how uFrame will generate it's boot loader, every element defined in the subsystem will have its controller registered, as well as any Registered Instances initialized.  This will also include any subsystems that are included into the attached subsystem.

In short, if a scene is going to use an element make sure its included in a subsystem that leads into the scene manager.
