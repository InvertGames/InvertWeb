Scene Transitions
Getting Started > Standard Graphs > Scene Managers > Scene Transitions

Scene Transitions allow you to easily set-up transitioning from one scene to another. 

Scene transitions require a registered instance in a connected subsystem.  The commands of this registered instance be available for invoking transitions to other scenes.

When you click on the '+' button these commands will be displayed in a list in order to add them as transitions. 

When the code is generated it will subscribe to the command and begin the transition when executed, just like any other command.

Here is an example of the Play transition as generated code in the FPSMainMenuManagerBase class.

GeneratedSceneTransition
Copy Code 
    public override void Initialize() {
        base.Initialize();
        FPSMenu.Play.Subscribe(_=> Play()).DisposeWith(this.gameObject);
    }
    public virtual System.Collections.Generic.IEnumerable<string> GetPlayScenes() {
        return this._PlayTransition._Scenes;
    }
   
    public virtual void Play() {
        GameManager.TransitionLevel<WavesFPSGameManager>((container) =>{container._WavesFPSGameManagerSettings = _PlayTransition;   PlayTransitionComplete(container); }, this.GetPlayScenes().ToArray());
    }
