Initialize{ElementName} Method
Getting Started > Element Graphs > Using Elements > Element Controllers > Initialize{ElementName} Method

The initialize Method in a controller can be used to initialize an Element's ViewModel (similar to how it might be initialized in the inspector of a View). It is a great place to subscribe to "Real Time" property changes or track a list of ViewModel Instances, which act similar to a Manager.

Every element controller has an Initialize{ElementName} method that can be overridden from the generated base class. For example:

InitializeMethod
Copy Code 
    public override void InitializeFPSWeapon(FPSWeaponViewModel fpsWeapon)
    {
        fpsWeapon.StateProperty
            .Where(p => p is Reloading) // Subscribe only to when the state is changed to reloading
            .Subscribe(r =>
            {
                // When we've entered the reloading state create a timer that executes a command invoking a transition.
                Observable.Timer(TimeSpan.FromSeconds(fpsWeapon.ReloadTime))
                    .Subscribe(l =>
                    {
                        fpsWeapon.Ammo = fpsWeapon.RoundSize;
                        ExecuteCommand(fpsWeapon.FinishedReloading);
                    });
            }).DisposeWith(fpsWeapon); // Make sure this is disposed with the weapon
    }
