Executing Commands From Controllers
Getting Started > Element Graphs > Using Elements > Element Controllers > Executing Commands From Controllers

One thing to note about commands on controllers, calling a command method directly will not invoke any subscribers.  Sometimes this may be intended behaviour, in order to make sure a command is invoked properly simply use one of the "ExecuteCommand" overloads available to all controllers. To demonstrate the difference examing the following:

Executing Commands
Copy Code 
public void Hit(EnemyViewModel enemy, DamagerViewModel damager) {
     // Won't notify subscribers
     DamagerController.PowerUp(damager);
     // Will notify subscribers
     ExecuteCommand(damager.PowerUp);
}
