Commands

Commands define a property of type ISignal on an element. These act as a delegate to a pre-initialized handler.The handlers are setup in the controllers initialize method to point to a method, this makes a controller a factory for a view-model.

Executing a command from a viewmodel.

MyViewModel.CommandName.OnNext(new {CommandName}Command() { });
